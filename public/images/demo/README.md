# Демо-ассеты витрины

Папка содержит вспомогательные ресурсы для демо-слоя витрин (маркет, кейсы, главная).

## Источники изображений

Для демо используются картинки, уже лежащие в репозитории:

- Оружие: `public/images/playstore/goods/csgo/weapon/*.png` (ak47, awp, m4a4, m4a1s, deagle, glock, usp, aug).
- Кейсы: `public/images/playstore/goods/csgo/case/*.png` (~45 обложек: chroma, hydra, falchion, dragonlore и др.).

Один визуал переиспользуется под разные имена скинов (например, `ak47.png` показывается и для AK-47 | Redline, и для AK-47 | Vulcan). Это нормальный приём для прототипа витрины.

## Маркировка демо-данных

В БД демо-записи отличаются по маркеру `steam_id` и `username` демо-продавцов:

- `steam_id` начинается с префикса `demo_` (например, `demo_7656110000000001`).
- `username` формата `demo_seller_N`.

Демо-категория кейсов — `Демо-кейсы` (строго такое имя).

Команда очистки:

```bash
php artisan demo:wipe
```

Удаляет только записи, привязанные к демо-продавцам и демо-кейсам. Картинки в `public/images/` не трогает.

## Добавление своих превью

Если нужно подкрепить витрину более разнообразными превью реальных скинов CS2, положите PNG в `public/images/demo/skins/` и расширьте каталог в `database/seeders/demo/DemoSkinCatalog.php`.
